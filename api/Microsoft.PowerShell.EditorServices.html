VariableContainerDetails

Container for variables that is not itself a variable per se. However given how VSCode uses an integer variable reference id for every node under the "Variables" tool window, it is useful to treat containers, typically scope containers, as a variable. Note that these containers are not necessarily always a scope container. Consider a container such as "Auto" or "My". These aren't scope related but serve as just another way to organize variables into a useful UI structure.
